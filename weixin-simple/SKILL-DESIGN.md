# 微信公众号艺术配图生成器

## 功能概述

基于 [canvas-design](https://skills.sh/anthropics/skills/canvas-design) 技能，为微信公众号创建艺术级配图。

## 使用场景

### ✅ 适合使用
- 公众号文章封面（艺术风格）
- 品牌推广海报
- 抽象概念配图
- 节日/活动专题图

### ❌ 不适合使用
- 需要展示具体产品/人物
- 信息图表/数据可视化
- 快速批量生成

## 工作流程

### 第一步：理解文章主题
- 用户输入文章主题和风格偏好
- 分析文章的情感基调和视觉方向

### 第二步：创建设计哲学
- 命名艺术运动（1-2个词）
- 阐述设计哲学（4-6段）
- 输出为 `.md` 文件

### 第三步：生成艺术配图
- 使用 Python 代码创作
- 输出为 `.png` 文件
- 微信封面建议尺寸：900*500px 或 1080*1080px

### 第四步：保存到工作目录
文件命名格式：`主题名称-发布日期-art.png`

## 设计哲学模板

### 干货教程类
**命名**: "Knowledge Clarity"（知识清晰）
**哲学**: 通过清晰的视觉层次和结构化的空间布局，让知识像光线一样穿透。使用几何形状构建框架，色彩克制而精准，每个元素都服务于理解。

### 观点评论类
**命名**: "Perspective Geometry"（视角几何）
**哲学**: 观点是有角度的，视觉应该体现这种张力。通过斜切、重叠、透视来表现不同观点的碰撞与融合。

### 案例分析类
**命名**: "Narrative Layers"（叙事层叠）
**哲学**: 案例是有层次的，通过层叠的透明色块和渐进的视觉节奏，展现故事的深度和时间性。

### 热点追踪类
**命名**: "Pulse Frequency"（脉搏频率）
**哲学**: 热点是动态的，通过重复的波纹、渐变的频率，展现信息的流动和能量。

## 技术限制

- 需要支持 Python 绘图库（matplotlib / PIL）
- 生成时间：30-60秒
- 输出格式：PNG（推荐）或 PDF

- `weixin-canvas-design`：生成艺术级配图
- 两者配合实现完整的公众号内容创作
