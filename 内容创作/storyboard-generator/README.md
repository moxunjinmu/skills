# 视频分镜脚本生成技能

基于元提示词 SOP 方法论的视频分镜脚本生成技能。

## 目录结构

```
storyboard-generator/
├── SKILL.md                    # 技能主文档
├── README.md                   # 本文件
└── references/                 # 参考案例
    └── seedance-storyboard-prompts.md  # 虫族入侵分镜脚本示例
```

## 使用方法

直接在对话中描述你的视频需求，AI 会自动：

1. **素材投喂**：理解你的项目背景和风格要求
2. **元指令分析**：拆分镜头、分配时间、生成提示词
3. **沙盒模拟**：提供测试建议和优化方案

## 示例

```
用户：我想做一个 60 秒的科幻短视频，故事是虫族入侵地球

AI 会自动：
- 分析故事结构
- 拆分为 6-8 个片段
- 生成适配 Seedance/Sora 的提示词
- 提供测试和优化建议
```
