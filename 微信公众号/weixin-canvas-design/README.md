# 微信公众号艺术配图生成器

基于设计哲学创建艺术级公众号封面和配图。

## 功能特点

- **两步创作流程**：设计哲学 → 视觉表达
- **7种文章类型**：覆盖常见公众号内容类型
- **博物馆级品质**：每个细节都经过精心打磨
- **无需 Python**：Claude 直接处理

## 目录结构

```
weixin_canvas_design/
├── SKILL.md              # 技能配置文件
├── README.md             # 项目说明
└── scripts/              # 脚本目录（预留）
```

## 文章类型与设计哲学

| 类型 | 设计哲学 | 视觉特征 |
|------|----------|----------|
| 干货教程 | Knowledge Clarity | 几何精度、清晰层次、系统化 |
| 观点评论 | Perspective Geometry | 角度张力、动态对比、视觉冲突 |
| 案例分析 | Narrative Layers | 层叠透明、时间流动、故事深度 |
| 热点追踪 | Pulse Frequency | 波纹重复、能量辐射、节奏感 |
| 科技前沿 | Future Logic | 极致几何、冷峻色彩、理性秩序 |
| 情感故事 | Emotional Resonance | 有机曲线、温暖渐变、柔和边缘 |
| 商务分析 | Structured Authority | 大胆分割、自信色彩、权威层次 |

## 使用方式

### 在 Claude Code 中使用

```
用户：帮我写一篇 AI 教育的文章，生成艺术风格封面

Claude：
1. 生成文章内容
2. 创建 "Knowledge Clarity" 设计哲学
3. 生成艺术封面图
```

### 输出文件

- 设计哲学：`主题名称-philosophy.md`
- 艺术配图：`主题名称-art.png`

## 与其他技能配合

### 配合 weixin-simple

```
weixin_simple → 文案生成 + 排版
weixin_canvas_design → 艺术封面
```

完整工作流：
1. 生成文章内容
2. 生成艺术封面
3. 组合发布

## 适用场景

### ✅ 适合

- 艺术风格公众号封面
- 品牌推广海报
- 专题活动图
- 抽象概念配图

### ❌ 不适合

- 需要展示具体产品/人物
- 信息图表/数据可视化
- 快速批量生成

## 技术规格

- 输出格式：PNG
- 推荐尺寸：
  - 横版封面：900×500px
  - 方版封面：1080×1080px
  - 竖版海报：1080×1920px

## 设计原则

1. **视觉哲学优于文字** - 信息通过设计传达
2. **空间优于拥挤** - 大量留白，元素呼吸
3. **克制优于过度** - 每个元素都有明确目的
4. **工艺优于速度** - 每个细节都精心打磨
