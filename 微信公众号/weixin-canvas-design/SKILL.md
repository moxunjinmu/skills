# 微信公众号艺术配图生成器

## 功能概述

本技能基于 [canvas-design](https://skills.sh/anthropics/skills/canvas-design) 的设计哲学理念，为微信公众号创建**艺术级**配图。

与普通图片生成不同，本技能通过以下两步流程创作：

1. **创建设计哲学** - 命名艺术运动，阐述美学理念
2. **视觉表达** - 使用 Python 代码将哲学转化为艺术作品

---

## 使用场景

### ✅ 适合使用

- **艺术风格封面**：需要抽象、艺术感的公众号封面
- **品牌推广海报**：公众号品牌形象设计
- **专题活动图**：节日、活动专题封面
- **抽象概念配图**：表达抽象概念或情感

### ❌ 不适合使用

- 需要展示具体产品/人物
- 信息图表/数据可视化
- 快速批量生成图片

---

## 工作流程

### 第一步：理解创作需求

1. 用户输入文章主题或配图需求
2. 确定文章类型和情感基调
3. 确定视觉风格方向

### 第二步：创建设计哲学

创建一个 `.md` 文件，包含：

1. **命名艺术运动**（1-2个词）
   - 示例："Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

2. **阐述设计哲学**（4-6段）
   - 空间与形式
   - 色彩与材质
   - 尺度与节奏
   - 构图与平衡
   - 视觉层次

### 第三步：生成艺术配图

使用 Python 代码创作艺术作品：
- 输出为 `.png` 文件
- 微信封面建议尺寸：900*500px 或 1080*1080px
- 90% 视觉设计 + 10% 必要文字

### 第四步：精细打磨

让作品达到博物馆级品质：
- 检查对齐和间距
- 优化色彩搭配
- 确保视觉平衡

### 第五步：保存到工作目录

文件命名格式：`主题名称-发布日期-art.png`

## 公众号文章类型的设计哲学

### 干货教程类

**命名**: "Knowledge Clarity"（知识清晰）

**哲学**: 知识应该像光线一样穿透迷雾。通过清晰的视觉层次和结构化的空间布局，让复杂的概念变得直观易懂。使用几何形状构建框架，色彩克制而精准，每个元素都服务于理解。网格系统的精度反映了知识的系统性，负空间的运用让内容呼吸。这是信息设计的最高形式 - 让看不见的思考过程变得可见。每一个对齐都是经过无数次调整的结果，每一个色彩选择都服务于认知的清晰。这是大师级的信息可视化设计。

### 观点评论类

**命名**: "Perspective Geometry"（视角几何）

**哲学**: 观点是有角度的，视觉应该体现这种张力和碰撞。通过斜切的线条、重叠的平面、透视的变形，展现不同观点的激烈对话。视觉元素不再水平排列，而是以动态的角度切入，创造戏剧性的空间关系。色彩对比强化观点的差异，几何形状的碰撞象征思想的交锋。每一个元素都在争夺注意力，每一个角落都暗含着张力。这是辩论的视觉化，是思想角力的空间表达。精心计算的不平衡创造出引人入胜的动态感，每一个角度都经过大师级的美学考量。

### 案例分析类

**命名**: "Narrative Layers"（叙事层叠）

**哲学**: 案例是有层次的，像考古地层一样藏着故事的深度。通过层叠的透明色块、渐进的视觉节奏、时间的空间化，展现故事的丰富维度。每一个图层都是故事的一个片段，透明度的变化让过去和现在对话。色彩从深沉到明亮，象征理解的深入。元素以有机的方式排列，像是记忆在空间中的沉淀。 typography 漂浮在不同的层次上，像是时间的回声。这是叙事设计的艺术，是让看不见的时间变得可见。每一个层次都经过精心调试，每一处透明都经过大师级的色彩校准。

### 热点追踪类

**命名**: "Pulse Frequency"（脉搏频率）

**哲学**: 热点是动态的，像脉搏一样有节奏地跳动。通过重复的波纹图案、渐变的频率密度、放射的能量线，展现信息的流动和传播。圆形的重复创造了节奏感，色彩的渐变象征能量的扩散。元素从中心向外辐射，像是一颗石子投入平静的湖面。动态的视觉效果传达了热点的时效性和紧迫感。每一次重复都是一次信息的强化，每一层扩散都是一次影响力的延伸。这是动态信息的视觉化，是让看不见的传播变得可见。每一个波纹都经过精心设计，每一次扩散都体现大师级的空间控制。

### 科技前沿类

**命名**: "Future Logic"（未来逻辑）

**哲学**: 科技是理性的极致，是逻辑的美学表达。通过极致的几何精度、冷峻的色彩系统、系统化的排列，展现科技的纯粹和力量。线条锋利如刀，色彩冷静如冰，每一个元素都服从于严格的逻辑秩序。网格系统不再只是工具，而是美学的核心。负空间被精确计算，色彩被理性分配。这是工程美学的极致，是功能与形式的完美统一。每一个像素都在正确的位置，每一个色彩都服务于整体。这是理性的诗学，是逻辑的视觉化表达。所有的细节都经过大师级的精确控制。

### 情感故事类

**命名**: "Emotional Resonance"（情感共鸣）

**哲学**: 情感是流动的，像水一样柔软而有力。通过有机的曲线形态、温暖的色彩渐变、柔和的边缘处理，营造情感的深度和温度。形状不再几何，而是流动的、有机的、有生命力的。色彩像调色盘上的颜料一样交融，创造出丰富的情感层次。负空间被温柔地填充，边缘被柔化处理。typography 像耳语一样轻柔地融入画面。这是情感的视觉化，是让看不见的感受变得可见。每一条曲线都经过精心勾勒，每一处渐变都经过大师级的色彩调试。

### 商务分析类

**命名**: "Structured Authority"（结构权威）

**哲学**: 商务需要权威感，需要通过视觉传递专业和可信。通过大胆的几何分割、自信的色彩块面、强烈的视觉层次，构建权威的视觉语言。线条刚直有力，色彩饱和坚定，布局清晰有力。每一个元素都传递着确定性，每一个角落都散发着专业感。这是商务美学的典范，是权威的视觉化表达。网格系统被严格执行，色彩系统被精心设计。所有的元素都服务于一个目标：建立信任。每一个细节都经过大师级的商业美学考量。

## 设计原则

### 核心理念

1. **视觉哲学优于文字描述**
   - 信息通过设计传达，而非文字解释

2. **空间优于拥挤**
   - 大量留白，让元素呼吸

3. **克制优于过度**
   - 每个元素都有明确目的

4. **工艺优于速度**
   - 每个细节都经过精心打磨

### 质量标准

- 看起来像花费数小时创作
- 每个元素位置都经过精确计算
- 色彩搭配和谐统一
- 视觉层次清晰有序

## 常见问题

### Q1：生成需要多长时间？

A：通常 30-60 秒，因为需要精细打磨每个细节。

### Q2：可以修改已生成的图片吗？

A：可以，本技能支持基于设计哲学进行迭代优化。

### Q3：适合什么类型的公众号？

A：特别适合艺术、文化、设计、科技等追求视觉品质的公众号。

### Q4：与普通 AI 绘图有什么区别？

A：普通 AI 绘图直接生成图片，本技能先创建设计哲学再创作，确保作品的艺术性和一致性。
