# 微信公众号违禁词检查

## 功能

提供完整的广告法违禁词库，用于公众号文章发布前的内容合规检查。

## 违禁词分类（11大类）

1. 与「最」有关（33个）— 最佳、最具、最优秀等
2. 与「一」有关（19个）— 第一、唯一、NO.1等
3. 与「级/极」有关（16个）— 国家级、顶级、极致等
4. 与「首/家/国」有关（18个）— 首个、独家、国家免检等
5. 与「品牌」有关（18个）— 大牌、领袖、之王等
6. 与「虚假」有关（16个）— 史无前例、永久、万能等
7. 与「权威」有关（10个）— 老字号、专家推荐等
8. 与「欺诈」有关（17个）— 点击领奖、秒杀、抢爆等
9. 与「时间」有关（13个）— 倒计时、仅限、随时涨价等
10. 与「迷信」有关（16个）— 招财进宝、转运等
11. 与「医疗」有关（大量）— 抗癌、减肥、杀菌等

## 使用方式

1. 读取 `{skill_root}/references/违禁词库.md`
2. 逐条检查文章内容是否包含违禁词
3. 标记命中的违禁词及所在位置
4. 提供替换建议

## 检查流程

```
文章内容 → 加载违禁词库 → 逐类扫描 → 输出报告
```

### 输出格式

```
违禁词检查报告：
- ⚠️ 发现 X 个违禁词
- 第N段：「最佳」→ 建议替换为「优质」
- 第N段：「第一」→ 建议替换为「领先」
- ✅ 其余内容合规
```

## 注意事项

- 某些词在特定语境下可能合规（如引用、学术讨论），需人工判断
- 违禁词库会随政策更新，建议定期检查是否需要补充
- 医疗类违禁词范围最广，健康类文章需特别注意
