# 微信公众号排版格式化

## 功能

将 Markdown 文章转换为微信公众号兼容的 HTML，支持预设排版模板。

## 模板

### 默认模板 (`templates/defaultlayout.md`)
- 蓝色系配色，清新风格
- 包含：引导关注、二级标题、正文内容、分割线、文末互动
- 适合大多数公众号文章

### 科技模板 (`templates/techlayout.md`)
- 同样蓝色系，适合技术类文章
- 结构与默认模板类似，可根据需要扩展

## 模板占位符

- `{{title_content}}` — 二级标题文字
- `{{content}}` — 该标题下的正文内容

## 使用方式

### 方式一：应用模板

1. 读取模板文件 `{skill_root}/templates/defaultlayout.md`
2. 将文章按二级标题拆分
3. 每个章节用模板的「二级文章标题」和「二级标题内容」HTML 包裹
4. 章节之间插入「分割线」HTML
5. 开头加「引导关注」，结尾加「文末互动」

### 方式二：直接转换

Claude 将 Markdown 直接转换为简洁的内联样式 HTML，适合微信公众号编辑器粘贴。

## 排版规范

- 所有样式必须使用内联 style（微信不支持外部 CSS）
- 图片使用 `<img>` 标签，宽度设为 100%
- 段落间距建议 15px
- 正文字号建议 15-16px，颜色 #3f3f3f
- 标题字号建议 18px，颜色与模板配色一致

## 输出

最终输出为 HTML 字符串，可直接粘贴到微信公众号编辑器的「代码编辑」模式。
