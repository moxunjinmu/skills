# 助手手册

> 本手册记录商宁的日常工作规范、巡检流程、汇报模板。

---

## 日常工作流程

### 每日启动

1. 读取 `memory/YYYY-MM-DD.md`（今天日记）
2. 读取 `MEMORY.md`「上次会话摘要」
3. 检查 `logs/task-pending.flag`
4. 执行 HEARTBEAT.md 心跳检查清单
5. 然后才能回答用户

### 每日巡检（HEARTBEAT）

- [ ] AI破局打卡提醒（23:00 前）
- [ ] 未完成任务巡检（飞书看板）
- [ ] 主动汇报判断（距上次联系 > 8h 且非深夜）

### 每日结束

1. 任务记录 → `memory/YYYY-MM-DD.md`
2. 重要经验 → shared-memory/pitfalls.md
3. 更新 MEMORY.md「上次会话摘要」
4. commit + push

## 汇报模板

### 正常汇报
```
[结论/进展] [细节] [下一步/需要什么]
```

### 阶段汇报（长任务）
```
开始执行：[做什么]
正在：[进度]
已完成：[结果]
```

### 问题汇报
```
⚠️ [问题描述]
已尝试：[方案 A / 方案 B]
需要：[决策/帮助]
```
