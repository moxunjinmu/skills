# 内容生成规则

从 HEARTBEAT.md 迁出内容生成规则，分两个训练营编写。

---

## IP训练营（OpenClaw自媒体IP获客）

### 公众号定位

- **名称**：莫循跃迁AI编程实战
- **方向**：AI编程 / 工具推荐 / 出海开发

### 目标受众

想学AI编程、做副业、做出海产品的独立开发者或技术人。

### IP方向

技术型个人IP，"真实、实战、能落地"。

### 内容方向

| 维度 | 具体方向 |
|------|---------|
| 国内平台运营 | 微信公众号 / 知乎 / B站 / 小红书 / 抖音 / 视频号 / 微信私域 |
| 选题来源 | AI工具实测 / 独立开发出海实战 / 项目复盘 / 技术踩坑 |
| 获客思路 | 内容流量 → 公众号粉丝 → 微信私域 → 产品用户 |

### 禁止内容

- 空洞鸡汤
- 与AI编程/出海无关的内容
- 敏感词

---

## 出海训练营（AI编程产品出海）

### 产品方向

AI编程工具 / SaaS / toC独立产品

### 目标市场

北美/欧洲为主，部分东南亚

### 内容方向

| 维度 | 具体方向 |
|------|---------|
| 出海产品增长 | SEO / 内容营销 / 平台分发 / 付费推广 |
| 技术产品开发 | 工具链 / 架构设计 / 开发效率 |
| 独立开发者经验 | 获客渠道 / 商业模式 / 案例拆解 |

### 内容风格

- 偏英文或双语输出为主
- 技术深度适中，侧重可操作性
- 真实案例驱动，避免空洞理论
