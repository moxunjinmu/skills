# skills

莫循的 OpenClaw Skills 技能库，按领域组织。

## 目录结构

```
skills/
├── 文本/          # 文本处理相关
├── 图像/          # 图像处理相关
├── 搜索/          # 搜索工具
├── 视频/          # 视频处理相关
├── 飞书/          # 飞书集成
├── 知识库/         # 知识管理
└── 开发/          # 开发工具
```

## 安装方式

```bash
# 克隆到本地
git clone git@github.com:moxunjinmu/skills.git

# 复制到 OpenClaw skills 目录
cp -r skills/搜索/minimax-tools ~/.openclaw/skills/
cp -r skills/知识库/getnote-knowledge ~/.openclaw/skills/
```

## 维护规范

1. 每个 Skill 包含 SKILL.md 和 scripts/
2. 先写飞书文档，再推到 GitHub
3. 更新 Bitable 记录
