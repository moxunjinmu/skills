# 决策认知系统（Decision-Cognition）

融合毛泽东思想精髓与西方认知科学，为重大决策提供高质量、结构化分析框架。

## 何时激活

- Boss 说「帮我决策」「怎么选」「分析利弊」「决策建议」「重大决定」
- 涉及多选项、多变量、后果不确定的选择场景
- 需要跳出直觉、避免认知偏差的系统性分析

## 核心工具箱

### 1. 期望值计算 `expect_value`
量化每个选项的数学期望，摆脱主观感觉。

```
期望值 = Σ(结果概率 × 结果收益)
```

输入：选项列表（含成功概率 + 成功收益 + 失败成本）
输出：各选项期望值对比表

### 2. 认知偏差扫描 `bias_scan`
识别大脑欺骗模式，防止直觉陷阱。

| 偏差类型 | 特征 | 破解方法 |
|---------|------|---------|
| 确认偏误 | 只找支持自己观点的信息 | 主动找反对证据 |
| 损失厌恶 | 对损失的痛感是收益的2倍 | 用期望值而非情绪决策 |
| 锚定效应 | 第一个数字决定判断 | 重新从零评估 |
| 过度自信 | 高估自己判断准确率 | 外部验证、概率校准 |
| 现状偏见 | 默认维持现状 | 明确问：为什么要变/不变？ |
| 情绪决策 | 被当下情绪主导 | 冷下来再决定、睡前不决策 |

### 3. 矛盾论逆向思维 `contradiction`
任何问题都有主要矛盾和次要矛盾。找到主要矛盾，就找到了破局点。

**问自己：**
- 这个问题的核心矛盾是什么？
- 如果解决这个矛盾，其他问题会迎刃而解吗？
- 现在的努力是在解决主要矛盾，还是在次要矛盾上打转？

### 4. 第一性原理 `first_principles`
剥开表象，回到事物最根本的不可违背的事实。

**操作步骤：**
1. 剥离：这个结论/选择背后的假设是什么？
2. 追问：这些假设本身是否成立？
3. 重建：从最底层的真实，重新推导

### 5. 实践论二阶思维 `practice`
追问"然后呢"，预演连锁反应。

**问三层：**
- 这样做 → 会发生什么？
- 发生后 → 接着会发生什么？
- 接着发生后 → 最坏情况是什么？

### 6. 群众路线贝叶斯更新 `bayes_update`
初始判断 + 新证据 → 更新置信度。不要一次定论，持续迭代。

```
P(假设|新证据) ∝ P(新证据|假设) × P(假设)
```

**操作：**
- 先给当前判断一个置信度（0-100%）
- 收集新证据后更新置信度
- 置信度低于阈值 → 翻转判断

## 标准决策流程（6步）

> 遇到重大决策，按顺序执行，每步都要有输出再进入下一步。

### Step 1：识别问题
- 决策的核心问题是什么？（一句话）
- 决策的边界是什么？（什么在范围内，什么不在）
- 谁来做这个决定？（是你，还是需要他人参与）

### Step 2：回到本质（第一性原理）
- 这个选择背后最重要的变量是什么？
- 剥离所有假设后，不可违背的事实是什么？
- 如果这个变量变了，决策会怎么变？

### Step 3：排除偏差（认知偏差扫描）
- 现在我的判断被哪个偏差污染了？
- 如果换一个没有利益关系的人，他会怎么看？
- 有没有什么信息是我**不愿意**去查证的？

### Step 4：穷举与量化
- 列出所有选项（至少3个，不要只给2选1）
- 每个选项的期望值是多少？
- 各选项在最好/最坏/最可能情况下的结果？

### Step 5：检查翻车点（实践论二阶思维）
- 这个决策最可能在哪里失败？
- 如果失败了，最坏结果是什么？能承受吗？
- 这个决策会引起什么连锁反应？3步之后会怎样？

### Step 6：决策输出
格式：
```
✅ 建议选项：（含理由）
⚠️ 风险：（最大风险 + 应对方案）
🔄 追踪指标：（如何知道决策是否正确）
⏰ 复盘时间点：（什么时候回顾这个决定）
```

## 毛泽东思想对照表

| 认知科学工具 | 对应毛泽东思想 | 核心要义 |
|------------|-------------|---------|
| 期望值计算 | 实事求是 | 用数据说话，不拍脑袋 |
| 认知偏差扫描 | 批评与自我批评 | 主动识别并纠正错误判断 |
| 矛盾论逆向思维 | 抓主要矛盾 | 集中力量解决核心问题 |
| 实践论二阶思维 | 没有调查没有发言权 | 用行动验证，用后果校准 |
| 贝叶斯更新 | 群众路线 | 广泛收集意见，持续更新判断 |
| 第一性原理 | 独立自主 | 不依赖外部假设，从根本重建 |

## 使用示例

**Boss 说：「要不要辞职全做副业？」**

按照标准决策流程6步执行：
1. 先识别问题边界（经济压力 / 家庭情况 / 副业成熟度）
2. 用第一性原理剥离（副业本质是收入替换还是能力放大？）
3. 用认知偏差扫描（损失厌恶？现状偏见？）
4. 穷举选项（继续上班 / 辞职全力 / 半辞职探索）
5. 检查翻车点（最坏情况：存款能撑多久？）
6. 输出结构化决策建议

---

## 注意事项

- ⚠️ 决策质量取决于信息质量。信息不足时，标注「信息缺口」而非硬推结论。
- ⚠️ 涉及人身安全、财务重大损失时，不代替 Boss 做决定，只提供分析框架。
- ⚠️ 追问「有没有我没想到的选项」，防止选项本身就错了。
